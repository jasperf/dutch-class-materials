# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Directory Is

This is **not a software project**. It is a personal archive of materials for Dutch language instruction.

## What's tracked in this repository

- **`lessons/`** — Per-quarter lesson plans (`week-XX/lesson-plan.md`) and session schedules
- **`notes/`** — Teaching notes and course overview
- **`handouts/`** — Student-facing handout materials

## What's gitignored (not in the repo)

- **`books/`** — Copyrighted textbooks (PDFs, Affinity Designer files)
- **`diaries/`** — Class diaries (may contain personal information)
- **`admin/`** — Contracts, receipts, and bank scans
- **`.vibe/`** — Local Mistral Vibe CLI config

## Typical Tasks Here

- Working with `lessons/` markdown files (lesson plans, schedules)
- Organizing or renaming documents
- Setting up a new course quarter
