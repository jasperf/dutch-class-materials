## **Grammatica**

### **1. Zullen (shall/will)**
Gebruikt voor: **suggesties** ("zullen we…?"), **aanbiedingen** ("zal ik…?"), **toekomstplannen**, **beloftes**

| **Enkelvoud** | **Meervoud** |
|---------------|--------------|
| ik **zal** | wij/we **zullen** |
| jij/je **zult/zal** | jullie **zullen** |
| u **zult/zal** | zij/ze **zullen** |
| hij/zij/het **zal** | |

**Voorbeelden:**
- **Zal ik** de tafel dekken? *(Shall I set the table?)*
- **Zullen we** morgen afspreken? *(Shall we meet tomorrow?)*
- Ik **zal** er om acht uur zijn. *(I'll be there at eight.)*
