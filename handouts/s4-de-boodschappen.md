## **Onderwerpen & Vocabulaire**

### **Boodschappen doen**
| **Nederlands** | **Engels** |
|----------------|------------|
| boodschappen doen | to do the shopping |
| Ik wil graag… | I'd like… |
| Heeft u…? / Hebben jullie…? | Do you have…? |
| een ons / een pond / een kilo | 100g / 500g / 1kg |
| Hoeveel kost dat? | How much does that cost? |
| Dat kost… euro | That costs… euros |
| betalen | to pay |
| het wisselgeld | the change |
| de bon / het bonnetje | the receipt |

## **Grammatica**

### **1. Meervoudsvormen (Plurals)**

| **Regel** | **Voorbeeld** |
|-----------|---------------|
| + **-en** (meeste woorden) | de man → de **mannen** |
| + **-s** (eindigt op -el, -em, -en, -er, -aar) | de tafel → de **tafels** |
| + **-\'s** (eindigt op a, o, u, i, y) | de foto → de **foto's** |
| Lange klinker verkort + **-en** | de stad → de **steden**; de weg → de **wegen** |

### **2. Modale werkwoorden + infinitief**
Artikel an das **Ende** des Satzes!

| **Nederlands** | **Engels** | **Voorbeeld** |
|----------------|------------|---------------|
| willen | to want | Ik wil graag een kilo appels **kopen**. |
| kunnen | can | Kunt u dat **herhalen**? |
| mogen | may | Mag ik dit **proberen**? |
| moeten | must / have to | Ik moet vandaag **boodschappen doen**. |
