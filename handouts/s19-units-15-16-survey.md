## **Full Course Grammar Summary**

| **Grammaticapunt** | **Kernregel** |
|-------------------|---------------|
| **zijn / hebben** | Volledige vervoeging uit je hoofd |
| **de / het** | Leer met elk zelfstandig naamwoord; verkleinwoorden altijd het |
| **Meervoud** | -en, -s, -'s, klinkerverkorting types |
| **Bijvoeglijke naamwoorden** | -e overal; uitzondering: een + het-woord |
| **Modale werkwoorden** | + infinitief aan het einde van de zin |
| **Ontkenning** | niet (werkwoorden/bijv. nawn.) vs. geen (zelfst. nawn.) |
| **Vragen** | Inversie: W + S; jij → je na inversie |
| **Tijd vertellen** | half X = X min 30! |
| **Scheidbare werkwoorden** | Voorvoegsel aan het einde in hoofdzin |
| **Wederkerende werkwoorden** | me/je/zich/ons |
| **Perfect tense** | hebben/zijn + ge…t/d; onregelmatige lijst |
| **Imperfect** | stam + -te/-de; onregelmatige lijst |
| **zullen** | aanbiedingen, suggesties, toekomst |
| **zouden** | voorwaardelijk, beleefde verzoeken |
| **Bijzinnen** | werkwoord aan het einde; want ≠ omdat |
| **Vergelijkingen** | even … als / meer … dan |
