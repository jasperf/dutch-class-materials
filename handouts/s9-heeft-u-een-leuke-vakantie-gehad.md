## **Grammatica**

### **1. Voltooid tegenwoordige tijd (Perfect Tense)**
**Structuur:** **hebben/zijn + voltooid deelwoord (aan het einde van de zin)**

**Regelmatige voltooide deelwoorden:**
1. Werkwoordstam = infinitief minus **-en**
2. Voeg **ge-** voorvoegsel toe
3. Einde: als de stam eindigt op een letter uit **'t kofschip** → voeg **-t** toe, anders → **-d**

| **Infinitive** | **Stam** | **Laatste letter** | **Voltooid deelwoord** |
|----------------|----------|-------------------|-----------------------|
| werken | werk | k ✓ | **gewerkt** |
| horen | hoor | r ✗ | **gehoord** |
| leven | leef | f ✓ | **geleefd** |
| wachten | wacht | t ✓ | **gewacht** |

**'t kofschip:** **t, k, f, s, ch, p, x**

### **2. Hebben vs. Zijn als hulpwerkwoord**

| **Gebruik *zijn*** | **Voorbeeld** |
|--------------------|---------------|
| Beweging naar een bestemming | Ik ben naar Parijs **gegaan** |
| Toestandsverandering | Het is warm **geworden** |

| **Gebruik *hebben*** | **Voorbeeld** |
|----------------------|---------------|
| Meeste werkwoorden (overgankelijke werkwoorden; doen/voelen/denken) | Ik heb gisteren veel **gewerkt** |

**Belangrijke onregelmatige voltooide deelwoorden (eerste batch):**
| **Infinitive** | **Voltooid deelwoord** |
|----------------|-----------------------|
| zijn | **geweest** |
| hebben | **gehad** |
| gaan | **gegaan** |
| komen | **gekomen** |
| doen | **gedaan** |
| zien | **gezien** |
| eten | **gegeten** |
| drinken | **gedronken** |
| bezoeken | **bezocht** |
| maken | **gemaakt** |
| wandelen | **gewandeld** |
